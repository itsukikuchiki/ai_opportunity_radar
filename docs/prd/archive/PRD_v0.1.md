# PRD v0.1

## Product definition
AI Opportunity Radar is a life-observation system that captures low-friction user signals, builds long-term memory around repeated patterns and frictions, and identifies promising AI automation opportunities.

## Core goals
- Observe real life with low friction
- Build memory around pattern / friction / desire / opportunity
- Produce weekly insights and AI opportunity suggestions

## Core modules
- Signal Capture
- Signal Understanding
- Memory Engine
- Insight Engine
- Opportunity Engine
