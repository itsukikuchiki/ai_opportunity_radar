# PRD v0.3

## Data flow
User input -> classification -> raw memory -> pattern/friction/desire update -> opportunity recalculation -> today output / weekly aggregation -> feedback loop

## Pages
- Today: quick capture, insight, pending question, daily best action, recent signals
- Weekly: key insight, top patterns, top frictions, best action, opportunity snapshot
- Opportunity Detail: why, evidence, score, recommendation, feedback
- Memory: patterns, frictions, desires, experiments
